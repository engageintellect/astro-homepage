# astro-homepage

## Description:

A simple landing page, build using [astro](https://astro.build).

# Features

- Astro
- TailwindCSS

## Getting Started:

```bash
git clone https://github.com/engageintellect/astro-homepage.git
```

```bash
cd astro-homepage
```

```bash
pnpm install
```

```bash
pnpm start
```
